# Victor Foster - AI/ML Engineer Portfolio

A modern, minimal, and professional portfolio website showcasing data analysis and AI/ML engineering projects.

## Overview

This portfolio website features a clean, light theme with gray and blue accents, designed to highlight professional projects and technical skills in a visually appealing and accessible manner.

## Features

- **Sticky Navigation** - Persistent top navigation for easy access to all sections
- **Responsive Design** - Fully responsive layout that works on desktop, tablet, and mobile devices
- **Project Showcase** - Dedicated pages for each featured project with detailed descriptions
- **Modern UI** - Clean, minimal design with smooth animations and transitions
- **Light Theme** - Professional light color scheme with blue and gray accents

## Structure

```
my-portfolio/
├── index.html              # Main landing page
├── styles.css              # Complete stylesheet
├── script.js               # JavaScript for smooth scrolling and animations
├── README.md               # This file
└── context/                # Content reference files
    ├── attention-grabber.md
    ├── contact-info.md
    ├── experience-skills.md
    ├── hard-skills.md
    ├── philosophy.md
    ├── project-links.md
    └── stategy,md
```

**Note:** Project notebooks are accessed directly from their respective project folders. Each notebook includes a sticky back navigation bar to return to the portfolio.

## Sections

### Main Page (index.html)
- **Hero Section** - Introduction with profile photo and call-to-action buttons
- **About Section** - Professional background and approach to AI/ML engineering
- **Projects Section** - Featured projects with links to interactive Jupyter notebooks
- **Skills Section** - Technical skills organized by category
- **Contact Section** - Contact information and social links

### Project Notebooks
Each project notebook opens in the same window and includes:
- **Sticky Back Navigation** - Persistent "Back to Portfolio" link at the top
- **Full Interactive Notebook** - Complete analysis with code, visualizations, and insights
- **Professional Formatting** - Clean, readable Jupyter notebook HTML export
